# 时间轴功能说明

## 功能概述

新增了时间轴视图功能，可以将任务按时间顺序在时间轴上展示，方便查看任务的时间分布。

## 主要特性

### 1. 视图切换
- 在应用顶部添加了视图切换按钮（位于进度指示器和设置按钮之间）
- 点击按钮可以在列表视图和时间轴视图之间切换
- 按钮图标会根据当前视图动态变化

### 2. 时间轴显示规则
- **有截止时间的任务**：按截止时间排序和显示
- **无截止时间的任务**：按创建时间排序和显示
- 时间轴从新到旧排列（最新的在顶部）

### 3. 时间轴元素

#### 时间点标记
- 已完成任务：绿色圆点 + 对勾图标
- 过期任务：红色圆点
- 高优先级任务：使用设置中的优先级颜色
- 普通任务：蓝色圆点

#### 时间标签
- 显示日期（今天、明天、昨天或具体日期）
- 显示具体时间（时:分）
- 有截止时间的任务会显示"截止"标签

#### 任务内容卡片
- 任务文本（已完成任务显示删除线）
- 所属分组标签
- 截止时间信息（如果有）
- 创建时间信息
- 快捷操作按钮（完成/删除）

### 4. 交互功能
- **双击任务**：编辑任务内容
- **右键菜单**：显示任务操作菜单（设置截止时间、删除等）
- **悬停效果**：卡片高亮、圆点放大
- **快捷按钮**：
  - 完成/取消完成按钮
  - 删除按钮

### 5. 响应式设计
- 支持不同屏幕尺寸
- 使用 clamp() 实现自适应字体和间距
- 支持夜间主题

## 技术实现

### 新增文件
- `src/components/TimelineView.vue` - 时间轴视图组件

### 修改文件
- `src/App.vue` - 添加视图切换功能和时间轴视图集成

### 核心逻辑
```typescript
// 时间轴项目生成
const timelineItems = computed(() => {
  return todos
    .map(todo => ({
      todo,
      displayTime: todo.deadline || todo.createdAt // 优先使用截止时间
    }))
    .sort((a, b) => b.displayTime - a.displayTime); // 从新到旧
});
```

## 使用方法

1. 点击顶部的时间轴图标按钮切换到时间轴视图
2. 在时间轴上查看任务的时间分布
3. 双击任务可以编辑
4. 右键点击任务可以打开操作菜单
5. 使用快捷按钮快速完成或删除任务
6. 点击列表图标按钮切换回列表视图

## 样式特点

- 垂直时间轴线，带渐变效果
- 时间标签在左侧，任务卡片在右侧
- 卡片使用毛玻璃效果（backdrop-filter）
- 平滑的动画过渡
- 支持深色主题
